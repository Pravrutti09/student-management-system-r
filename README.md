# Student Management System (R)

This project is a simple Student Management System developed using the R programming language. The goal of the application is to manage student records in an organized way through a console-based interface.

The program allows users to add new student information, update existing records, delete students, and view all stored student details.

## Features

Add new student records  
Edit existing student information  
Delete student records  
View all students in the system  
Simple menu-driven interface  

## Technologies Used

R Programming  
Lists and Functions  
Basic Data Management  

## How to Run

1. Open the script in RStudio or any R environment.
2. Run the program.
3. Use the menu options to manage student records.

## Purpose of the Project

This project was created to practice programming in R and understand how data structures like lists can be used to manage and organize information in a simple management system.
